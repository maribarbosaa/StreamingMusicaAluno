# StreamingMusicaAluno

Aplicação simples de streaming de música desenvolvida com **Programação Orientada a Objetos**. O sistema permite gerenciar músicas, playlists e usuários, simulando funcionalidades básicas de uma plataforma de streaming.

# Estrutura

* **Musica**: dados de uma música
* **Playlist**: coleção de músicas
* **Usuario**: gerenciamento de playlists
